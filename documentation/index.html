HTML Structure

The template is based on Bootstrap Framework - http://getbootstrap.com/
The default Bootstrap grid system utilizes 12 columns, making for a 1170px wide depending on your viewport.
Below 480px viewports, the columns become fluid and stack vertically.
 
Create a .row and add the appropriate number of .col-* columns.
  
<div class="row">
        
       { Level 1 column }

        <div class="row">
            <div class="col-6"> { Level 2 } </div>
            <div class="col-6"> { Level 2 } </div>
        </div>

</div>

If you need more information, please visit this site: http://getbootstrap.com/css/#grid

JavaScript

jQuery - is a Javascript library that greatly reduces the amount of code that you must write.
For more information, please visit http://www.jquery.com/
 
All the JavaScript libraries and files are included at the bottom of every HTML page like this: 
<!-- Jquery JS -->
<script src="js/jquery.min.js" type="text/javascript"></script>    
<!-- Scroll Animation JS -->
<script src="js/aos.min.js" type="text/javascript"></script>        
<!-- Bootstrap JS -->
<script src="js/bootstrap.min.js" type="text/javascript"></script> 
<!-- Custom JS -->
<script src="js/custom.js" type="text/javascript"></script>       
<!-- Owl Carousel JS -->
<script src="js/owl.carousel.min.js" type="text/javascript"></script>   
<!-- Popper JS -->
<script src="js/popper.min.js"></script>

 

Fonts

To change the embedded font, please take a look in the head part of sass/_config.css: Note: You need a sass compiler to have it intepreted in css, you can achieve that with vscode live sass extension
  
@import url("https://fonts.googleapis.com/css2?family=Poppins&display=swap");

 

To change the font, first go to http://www.google.com/webfonts choose a font and add the generated code below the embedded link
 
Then edit text-color or background-color in sass/_variables.scss
  
$primary-color: #EA3F5A;
$secondary-color: #162447;
$tertiary-color: #1B1B2F;
$white: #fff;

$primary-text-color: $tertiary-color;
$secondary-text-color: $white;
$tertiary-text-color: #f4f4f4;
$title-text-color: $secondary-color;

$bg-color: #E5E5E5;

Sliders

Owl Carousel slider is the slider you will find in the content. It's an essay to use lighweight carousel with some decent features.
 
Each Section slider has class 'sliders' and each slide has '<div class="owl-carousel owl-theme">' wrapper for easy find where slide html code start. If you wont remove some slide just remove div with all content like this '<div class="item">'
  
<!-- Sliders -->
<div class="owl-carousel owl-theme">
  <div class="item">
    <div class="devugo-card">
      <div class="content">
        <div class="img">
          <img src="img/testimonial1.png" />
        </div>
        <span class="float-left">"</span><br/>
        <p>Debby is a skilled designer who pt in efforts to produce great
                  products. I recommend her for your projects.  <span>"</span>
        </p>
        <h5>Anthony B</h5>
      </div>
    </div>
  </div>
  <div class="item">
      ...
  </div>
  <div class="item">
      ...
  </div>
 
If you want change option for sliders you need to open js/custom.js and find '// Owl Carousel Start' function and check out owl carousel documentation for editing the options
 
  
// Owl Carousel Configuration
$(document).ready(function(){
  $('.owl-carousel').owlCarousel({
      loop:true,
      margin:30,
      nav:true,
      autoplay:true,
      autoplayTimeout:5000,
      autoplayHoverPause:true,
      responsive:{
          0:{
              items:1
          },
          600:{
              items:2
          },
          1000:{
              items:3
          }
      }
  })
});

You can find more documentation here https://owlcarousel2.github.io/OwlCarousel2/index.html
